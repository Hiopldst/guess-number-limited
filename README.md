# Guess the Number (Limited Tries)

A simple console game where the computer picks a random number between 1 and 20, and you have 3 attempts to guess it. After each wrong guess, the program tells you if the target number is higher or lower.

## How to play
1. Run the script.
2. Enter a number between 1 and 20.
3. You have 3 attempts to guess the correct number.
